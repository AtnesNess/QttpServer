Detailed Description

EventEmitter class.

Member Typedef Documentation

typedef void* native::EventEmitter::listener_t

addListener(), on(), and once() function returns listener_t object which is just an untyped pointer. If you're going to remove a listener from EventEmitter, you have to save this returned value.

Constructor & Destructor Documentation

native::EventEmitter::EventEmitter ( ) [protected]

To create an EventEmitter object, you must define a new class inheriting from EventEmitter. Before you can call addListener() or emit() function, you have to register the events that you want to accept. Typically, you can register events in your class contructor.
